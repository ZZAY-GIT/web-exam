# АИС «Электронная библиотека»

Веб-приложение для ведения реестра книг, поиска/фильтрации и публикации рецензий.

## Технологический стек
- Python (Flask, Flask-Login, SQLAlchemy)
- PostgreSQL
- Docker / Docker Compose

## Быстрый запуск (Docker)

Запустите контейнеры из корневой директории проекта:
```bash
docker compose up --build -d
```

Приложение будет доступно на порту `5012`: [http://localhost:5012/](http://localhost:5012/)

Доступные пользователи (для авторизации):
- Администратор: `admin` / `admin`
- Модератор: `moderator` / `moderator`
- Обычный пользователь: `user` / `user`

## Запуск локальных тестов

Перейдите в папку приложения и запустите тесты:
```bash
cd app
python -m unittest test_app.py
```
